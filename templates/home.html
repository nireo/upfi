upfi - file hosting service

A minimal file hosting service to store all your files!

Login Register
A simple interface
Upfi features a clean and simple interface to manage files. Everything from deleting, updating, creating to downloading files is supported!
Open source
Upfi is fully built with Go, and all of the source files are publicly available on github.
Easy to setup
There are instructions on the github so that deployment is as easy as possible. Anyone can do it!
Secure
All files held on the server are immediately encrypted, such that the server owner cannot open your private files!
